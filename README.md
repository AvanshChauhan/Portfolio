# 🚀 React + Vite Portfolio

A fast, minimal React setup powered by Vite — built for performance and smooth UI.

🌐 **Live Demo:** https://portfolio-two-omega-c0ike0fv0c.vercel.app/

---

## ⚡ Tech Stack

* React
* Vite
* JavaScript
* SCSS / CSS
* GSAP (for animations)

---

## ✨ Features

* Clean and responsive UI
* Smooth animations
* Fast load with Vite
* Component-based structure
* Interactive terminal-style elements

---

## 🛠️ Setup

```bash
npm install
npm run dev
```

---

## 📦 Build

```bash
npm run build
```

---

## 🔌 Plugins Used

* @vitejs/plugin-react (Oxc)

## 📌 Notes

* Built as a personal portfolio
* Focused on UI/UX + performance
* No unnecessary bloat — simple and effective
